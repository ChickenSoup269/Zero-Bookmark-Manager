|                    | + Over 10 professional themes including Tokyo Night and Monokai |
|                    | + Custom typography with Nerd Fonts support                     |
| Smart Management   | + Raindrop style sidebar with nested folder structure           |
|                    | + Full drag and drop support for folders and bookmarks          |
|                    | + Automatic duplicate detection and cleaning                    |
| AI and Interaction | + AI assistant for natural language command control             |
|                    | + Privacy focused support for Local AI like Ollama              |
|                    | + Integrated web preview modal for instant site viewing         |
| Advanced Tools     | + Link health check to identify dead or suspicious URLs         |
|                    | + Detailed visit count tracking for usage analytics             |
|                    | + Instant QR code generation for mobile sharing                 |
| System Features    | + Chrome Side Panel integration for persistent access           |
|                    | + Multilingual interface in English and Vietnamese              |
|                    | + Secure data export and import in JSON HTML and CSV            |
